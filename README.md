bebi103
===============================

version number: 0.0.1
author: Justin Bois

Overview
--------

Python utilities foor the Caltech course BE/Bi 103: Data Analysis in the Biological Sciences.

Installation / Usage
--------------------

To install use pip:

    $ pip install bebi103


Or clone the repo:

    $ git clone https://github.com/justinbois/bebi103.git
    $ python setup.py install
